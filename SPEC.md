# NBA球员综合实力排名工具 - 产品规范

## 1. 项目概述

### 项目名称
**NBARankPro** - 现役NBA球员综合实力排名工具

### 核心功能
一个多维度的现役NBA球员评估和排名系统，支持自定义权重评分、球员对比、数据可视化和筛选排序功能。

### 目标用户
- NBA球迷
- 篮球分析师
- 梦幻篮球玩家
- 篮球数据爱好者

## 3. 数据结构规范

### 球员基础信息
```javascript
{
  id: string,
  name: string,
  number: number,
  team: string,
  position: "PG" | "SG" | "SF" | "PF" | "C",
  height: number, // cm
  weight: number, // kg
  age: number,
  experience: number, // 球龄
  draft: {
    year: number,
    round: number,
    pick: number
  },
  salary: string
}
```

### 评分维度 (0-100)
```javascript
{
  // 进攻能力
  scoring: number,        // 得分能力
  efficiency: number,    // 得分效率
  ballHandling: number,  // 持球进攻
  offBall: number,       // 无球能力
  clutch: number,        // 关键球能力

  // 防守能力
  defense: number,       // 总体防守
  perimeter: number,     // 外线防守
  interior: number,      // 护框/内线
  rebounding: number,    // 篮板
  steals: number,        // 抢断
  blocks: number,        // 盖帽

  // 组织能力
  playmaking: number,   // 传球视野
  turnover: number,      // 失误控制 (反向)
  leadership: number,    // 带动队友/领导力

  // 身体与竞技状态
  durability: number,    // 耐操性/出勤率
  athleticism: number,   // 运动能力
  stamina: number,        // 比赛强度

  // 篮球智商
  basketballIQ: number,  // 战术执行
  decisionMaking: number,// 关键时刻决策
  impact: number         // 领导力/影响力
}
```

### 进阶数据
```javascript
{
  per: number,           // 球员效率值
  ws: number,            // 胜利贡献值
  bpm: number,           // 场均正负值
  vorp: number,          // 可替代球员价值
  trueShooting: number,  // 真实命中率
  usage: number          // 使用率
}
```

## 4. 功能模块规范

### 4.1 球员信息库
- 展示所有现役球员卡片列表
- 显示核心信息：头像(占位图)、姓名、球队、位置、综合评分
- 点击展开详情面板

### 4.2 排名系统
- **总榜**: 所有球员按综合评分排序
- **分位置榜**: PG/SG/SF/PF/C 各位置Top10
- **分年龄段**: 25岁以下、26-30、31+
- **分球队榜**: 各球队内部排名

### 4.3 评分系统
默认权重配置:
- 进攻能力: 30%
- 防守能力: 25%
- 组织能力: 20%
- 身体状态: 15%
- 篮球智商: 10%

### 4.4 数据可视化
- **雷达图**: 六边形能力图展示球员多维能力
- **进度条**: 各维度评分可视化
- **排名列表**: 带序号、评分、球队标签的列表

### 4.5 筛选与排序
- 按位置筛选 (可多选)
- 按球队筛选
- 按年龄范围筛选
- 按单项能力排序

### 4.6 球员对比
- 支持2-4名球员同屏对比
- 雷达图叠加对比
- 数据表格并排显示

### 4.7 自定义权重
- 滑块调节各维度权重 (0-100%)
- 实时重新计算排名
- 保存/加载权重方案

### 4.8 收藏功能
- 收藏球员到个人列表
- 生成个人Top10/TOP20
- 本地存储持久化

## 6. 技术实现

### 技术栈
- **前端**: 纯HTML5 + CSS3 + Vanilla JavaScript
- **图表**: Canvas API (自定义雷达图)
- **存储**: LocalStorage (收藏、权重)
- **图标**: Lucide Icons (CDN)
- **字体**: Google Fonts (Orbitron, Inter)
